Python - Introduction

Before you begin your journey into Python, we need to know what it is first. Simply speaking, Python is a programming language, a language that humans use to talk to computers.

This means that Python has many uses. For example, when talking to a computer, we can use Python to tell the computer to do math for us, or read off a grocery list.

And, because of these abilities, Python also has many real-world applications, for example, Google often uses Python. So, if you want a job in the tech industry, Python is a good starting point.
