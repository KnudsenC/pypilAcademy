Python - The Print Statement

In Python, one of the most important functions is the print statement.

Print is the command that tells Python to tell us something.

For example, if we want Python to tell us 5, all we have to do is type print(5) in Python. Give it a try, put any number in the print statement.

Python can even do math equations!

If we want Python to tell us the answer to 6 + 5, all we have to do is type print(6 + 5), and Python will tell us 11. We can do other equations as well, try print(12 - 6)!

Here are all the basic math operators in Python:

  • + = addition
  • - = subtraction
  • * = multiplication
  • / = division
  • ** = to the power of

Lastly, Python can also print words. To do this, just put the word or words you want inside quotation marks, these things: “ ”, and then put the quoted word or words inside the print statement, like so: print(“word”).
